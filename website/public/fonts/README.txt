Drop the Rabar_022 font file here to enable it for Arabic & Kurdish text:

  Rabar_022.woff2   (preferred)
  Rabar_022.ttf     (fallback)

Until a file is present, the site automatically uses Vazirmatn (a high-quality
Arabic/Kurdish web font) so Arabic and Kurdish already render correctly.

Source for the font: https://www.kurdfonts.com/font-info/531
(Convert the .ttf to .woff2 with any online converter for best performance.)
